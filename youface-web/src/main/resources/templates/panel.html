Aviso de privacidad y proteccion de datos

Los datos biometricos (fotografias faciales) que proporciones seran utilizados exclusivamente para tu identificacion durante los examenes presenciales mediante el sistema YOUFACE.

Una vez finalizados todos tus examenes, tus datos biometricos seran eliminados de forma permanente e irrecuperable de nuestros servidores. Ningun dato facial sera compartido con terceros ni utilizado para ningun otro fin. El tratamiento de estos datos se realiza conforme al Reglamento General de Proteccion de Datos (RGPD).
